xml

XML stands for eXtensible Markup Language which was designed to store and transport data was designed to be both human- and machine-readable.

Xpath

XPath is a major element in the XSLT standard and can be used to navigate through elements and attributes in an XML document.

XSLT-Transformation

XSL (eXtensible Stylesheet Language) is a styling language for XML and XSLT stands for XSL Transformations.This tutorial will teach you how to use XSLT to transform XML documents into other formats (like transforming XML into HTML).

XML Formatting objects

XSL Formatting Objects, or XSL-FO, is a markup language for XML document formatting which is most often used to generate PDFs.
